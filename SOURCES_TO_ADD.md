## How to Add These:

1. **For Websites**: Use "Manage Sources" → "Add Source" → Select "website" → Add URL and title
2. **For PDFs**: Use "Manage Sources" → "Add Source" → Select "pdf" → Upload file (if you have it)
3. **For Books**: Use "Manage Sources" → "Add Source" → Select "book" → Add title, author, ISBN → Add key notes in content field
4. **For Papers**: If you have PDF, upload it. Otherwise, add as "research-paper" type with URL/citation
